# steam-market-analysis
This project provides a comprehensive data-driven exploration of the Steam gaming ecosystem. By analyzing a dataset of almost 85,000 titles, I identified the statistical "Success Formula" that distinguishes market hits from average releases.

The project uses a dataset originally sourced from Google BigQuery. For accessibility, the processed data is available in steam_data_cleaned.csv. To run the analysis, ensure the CSV file is in the same directory as the notebook.
The dataset is provided in a ZIP archive due to file size limits.
